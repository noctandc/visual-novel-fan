# Visual Novels Warriors - APK Repository

Ce repository regroupe les versions **APK** de romans visuels (visual novels) basés sur des **fanfictions Warriors**.  
Il inclut aussi bien mes propres histoires que celles d’autres auteur/autrice.s, adaptées avec leur accord.

**Projet non officiel - non commercial - réalisé par un fan pour des fans.**

## Objectifs du projet

Ce repository sert à plusieurs fins :

- **Adapter des fanfictions Warriors** en romans visuels interactifs  
- **Expérimenter la narration visuelle**, la mise en scène et le choix interactif  
- **Constituer un portfolio développeur**  
- Améliorer mes compétences en :
  - écriture interactive
  - design d’interface
  - illustration et direction artistique
  - programmation
  - export et déploiement d’applications Android (APK)

## Téléchargement & installation

1. Téléchargez l’APK correspondant à l’histoire voulue.  
2. Sur votre téléphone Android :
   - ouvrez l’app *Fichiers* ou *Téléchargements*  
   - lancez l’APK  
   - acceptez l’installation depuis des sources externes (si demandé)  
3. Le roman visuel apparaîtra dans vos applications.

>  Aucun APK de ce repository ne contient de publicité, traçage ou contenu commercial.

## Contributions des auteur.s/autrice.s

Chaque adaptation de fanfiction est réalisée **avec l’autorisation explicite de l’auteur/autrice original.e**.  
Les scénarios, concepts, personnages ou illustrations qui leur appartiennent restent leur propriété.

Je ne modifie jamais une histoire sans l’accord de son auteur.
